# 🎧 Apps1 - Radio & Podcast streaming Android Application

Aplikasi Android untuk menampilkan daftar podcast dan podcaster menggunakan REST API dan dummy data fallback.

---

## 🇮🇩 Deskripsi (Indonesia)

Apps1 adalah aplikasi Android berbasis Kotlin yang menampilkan daftar podcast dalam dua tampilan:
- Horizontal list (Podcast populer)
- Grid list (Podcaster)

Aplikasi ini menggunakan:
- Retrofit untuk REST API
- RecyclerView untuk list
- ViewBinding
- Dummy data sebagai fallback jika API gagal

---

## 🇬🇧 Description (English)

Apps1 is an Android application built with Kotlin that displays podcast data in two layouts:
- Horizontal list (Featured podcasts)
- Grid list (Podcasters)

This app uses:
- Retrofit for REST API
- RecyclerView for lists
- ViewBinding
- Dummy data fallback when API fails

---

## ✨ Features

- 📻 List Podcast Horizontal
- 👤 List Podcaster Grid
- 🔍 Detail Podcast Screen
- 🌐 REST API Integration
- 📦 Dummy Data Fallback
- ⬅️ Back Navigation
- ⚡ Fast Loading RecyclerView

---

## 🛠 Tech Stack

- Kotlin
- Android Studio
- RecyclerView
